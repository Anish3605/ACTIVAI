🦾 ACTIVAI Rehabilitation Dashboard
ACTIVAI is a clinical-grade, single-page web application designed to track and evaluate patient neuromuscular recovery following bionic prosthetic integration.

The platform aggregates complex electromyography (EMG) telemetry and therapeutic exercise data into a highly scannable, interactive interface tailored for physiotherapists and patients.

🚀 Key Features
Dynamic Clinical Summary: Instantly computes critical longitudinal KPIs including total completed sessions, average exercise accuracy, current overall progress, and real-time EMG quality scores.

Rich Data Visualizations: Integrated interactive charts mapping performance improvements, micro-level exercise execution accuracy, and muscle fatigue thresholds across chronological training cycles.

AI Clinical Recommendation Engine: A rule-based analytics system that monitors performance metrics to automatically flag excessive muscular fatigue, suggest repetition progression adjustments, and verify EMG sensor placement tracking.

Interactive Session History: An accessible, expandable ledger system enabling clinicians to seamlessly drill down into granular exercise-specific telemetry matrices.

Live-Data Simulator & Dark Mode: Built-in simulation tool to demonstrate real-time incoming session data telemetry, alongside an instant theme switch toggle for varied clinical lighting conditions.

🛠️ Tech Stack
Frontend Architecture: Single-Page Vanilla JavaScript (ES6+)

UI Framework & Styling: Tailwind CSS (Fluid responsive design, Accessible typography, Dark Mode native)

Data Visualization Engine: Chart.js
